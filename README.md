# 📌 Barcode Attendance Web App (FOSS-Based)

A modern, dark-themed barcode attendance system built using HTML, CSS, and JavaScript.

## 🚀 Features
- Barcode scanning (camera + input)
- Local storage (JSON format)
- Admin login (basic)
- Recent scan logs
- Dark modern UI

## 🧠 FOSS Logic
- No backend required
- Uses browser localStorage
- Fully open and editable logic

## 📂 How It Works
1. Open index.html
2. Scan barcode
3. Data is saved with timestamp
4. View logs instantly

## 📌 Future Scope
- Excel export
- Cloud integration
- Advanced admin dashboard
